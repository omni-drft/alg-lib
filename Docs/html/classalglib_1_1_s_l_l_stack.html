Detailed Description

template<typename T>
class alglib::SLLStack< T >

SLLStack is a stack implementation that uses a singly linked list as a base structure. It allocates memory on the heap, so the stack can grow dynamically.

Template Parameters
Ttype of data stored on the stack.
