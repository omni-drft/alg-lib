Detailed Description

template<typename T, size_t capacity>
class alglib::ArrayStack< T, capacity >

Template based stack implementation that uses an array as a base structure. It doesn't allocate memory on the heap, but it has a fixed capacity that has to be defined at compile time.

Template Parameters
Ttype of data stored in stack
capacitymax capacity of stack (size of array)
